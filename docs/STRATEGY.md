# Ultron-OS — Strategy

## Mission
A voice-first, local-first personal AI you completely own: a pretrained LLM brain that knows YOUR data (RAG + growing memory), learns any skill you feed it, speaks and listens, has hands on your devices, and acts only behind a human-in-the-loop approval gate.

## Why it's different
| | ChatGPT/Claude/Gemini | Ultron-OS |
|---|---|---|
| Data | their servers | your machine |
| Memory | session/siloed | grows forever, searchable |
| Skills | frozen at training | feed it any PDF/URL/video |
| Devices | none | laptop + phone + browser |
| Trading | none | advice + paper + gated live |
| Voice | cloud, turn-based | local, streaming, interruptible |
| Cost | $20+/mo | $0 default, open-source |

## The moat (never violated)
Local-first · model-agnostic · open-standard connectors (MCP) · solo-optimized · free-by-default · skill-first · money/contact code stays gated + opt-in.

## Operating principle
Grow intelligence through **data, memory, and skills** — not model retraining. Pretrained LLMs made smart about *your* world.
